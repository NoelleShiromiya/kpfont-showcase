朝鲜民主主义人民共和国字体展示

千里马（KP CheonRiMa）

哥特体（黑体）。源自伟大领袖金日成同志发起的“千里马运动”，象征快速建设国家的精神。主要用于出版物、书籍、标语和海报的标题及正文。
以下是导入“KP CheonRiMa”字体的CSS代码： 
@font-face {
	font-family: 'KP CheonRiMa';
	src:url('https://noelleshiromiya.github.io/kpfont-showcase/fonts/KP-CheonRiMa.woff2') format('woff2');
	font-weight: 400;
}
@font-face {
	font-family: 'KP CheonRiMa';
	src:url('https://noelleshiromiya.github.io/kpfont-showcase/fonts/KP-CheonRiMa-Light.woff2') format('woff2');
	font-weight: 300;
}
